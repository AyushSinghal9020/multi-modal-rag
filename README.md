This is a Multi-Modal RAG Chatbot, you can chat with your documents 

* Supported Documents
  * Docs
    * PDF
    * CSV
    * PPTX
    * DOCX
  * Image
    * JPEG
    * JPG
    * PNG
  * Video
    * MP4
  * Audio
    * MP3
   
Steps to proceed 

* Clone this repo

Run
```
pip install -r requirements.txt
```

* Put your files in the `upload folder`

Run 
```
python app.py
```
